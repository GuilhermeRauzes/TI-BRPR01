# 📦 Sistema de Inventário TI-BRPR01

Aplicação web focada em resolver o problema de controle e rastreamento de equipamentos operacionais dentro de um site logístico. O sistema permite um CRUD completo de dispositivos, facilitando a vida do Field Service.

### 🎯 Funcionalidades
* Cadastro e controle de Handhelds, Impressoras e Notebooks.
* Painel de visualização rápida do status de cada equipamento.
* Interface simplificada para agilizar o atendimento no balcão de TI.

### 🛠️ Tecnologias
* Python (Flask)
* HTML/CSS/JS para a interface
